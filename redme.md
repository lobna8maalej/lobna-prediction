# Power BI Prediction Dashboard

This project is a Business Intelligence and Predictive Analytics dashboard developed using Power BI and Python.

The dashboard provides interactive data visualization, KPI tracking, and sales prediction features to help businesses analyze performance and make data-driven decisions.

## Features

* Interactive Power BI dashboards
* Sales and revenue analysis
* Predictive analytics using Python models
* KPI monitoring
* Data visualization
* Business performance insights

## Technologies

* Power BI
* Python
* Pandas
* Machine Learning
* Excel / CSV Data Sources
